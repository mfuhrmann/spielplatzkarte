Grundsatz

Diese Anwendung erhebt keine Nutzerkonten, setzt keine Tracking-Cookies und führt keine Analyse des Nutzerverhaltens durch.

Kartendaten

Die Anwendung verwendet Geodaten aus OpenStreetMap, die unter der Open Database Licence (ODbL) bereitgestellt werden. Die Daten sind öffentlich zugänglich; es werden keine personenbezogenen Daten übertragen.

Standortsuche (Nominatim)

Für die Ortssuche wird die API Nominatim abgefragt. Dabei wird der eingegebene Suchbegriff übertragen. Es werden keine weiteren Nutzerdaten weitergegeben.

Fotos (Panoramax)

Sofern zu einem Spielplatz Fotos verfügbar sind, werden diese vom Dienst Panoramax geladen. Beim Aufruf dieser Inhalte gelten die Datenschutzbestimmungen von Panoramax.

Standortbestimmung

Die Standortfunktion des Browsers wird nur auf ausdrücklichen Wunsch der Nutzenden aktiviert. Der ermittelte Standort wird nicht gespeichert und nicht an Dritte weitergegeben.
